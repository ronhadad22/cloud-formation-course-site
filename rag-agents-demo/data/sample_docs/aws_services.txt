AWS Services Overview

Amazon Web Services (AWS) is the world's most comprehensive and broadly adopted cloud platform, offering over 200 fully featured services from data centers globally. Millions of customers—including the fastest-growing startups, largest enterprises, and leading government agencies—are using AWS to lower costs, become more agile, and innovate faster.

Compute Services:
- Amazon EC2: Secure and resizable compute capacity in the cloud
- AWS Lambda: Run code without thinking about servers
- Amazon ECS: Highly secure, reliable, and scalable way to run containers
- Amazon EKS: Managed Kubernetes service

Storage Services:
- Amazon S3: Object storage built to retrieve any amount of data from anywhere
- Amazon EBS: Easy to use, high performance block storage
- Amazon EFS: Simple, serverless, elastic file storage
- AWS Backup: Centralized backup across AWS services

Database Services:
- Amazon RDS: Managed relational database service
- Amazon DynamoDB: Fast and flexible NoSQL database service
- Amazon Aurora: MySQL and PostgreSQL-compatible relational database
- Amazon ElastiCache: In-memory caching service

AI/ML Services:
- Amazon Bedrock: Build and scale generative AI applications
- Amazon SageMaker: Build, train, and deploy machine learning models
- Amazon Rekognition: Image and video analysis
- Amazon Comprehend: Natural language processing service

Networking:
- Amazon VPC: Isolated cloud resources
- AWS Direct Connect: Dedicated network connection to AWS
- Amazon CloudFront: Global content delivery network
- Elastic Load Balancing: Distribute incoming traffic

Security:
- AWS IAM: Identity and access management
- AWS KMS: Managed encryption keys
- AWS WAF: Web application firewall
- Amazon GuardDuty: Threat detection service
